Advanced Plot

 A função Advanced Plot foi desenvolvida com base na ferramenta web que possibilita a geração de gráficos com atributos disponíveis no banco de dados do INCT.



 A geração dos gráficos dependem da entrada dos filtros que são separados em três sessôes:

  • Data e Hora
  • Estação, Atributo, Elevação e Satélites
  • Estilos do gráfico

Data e Hora

 A primeira sessão, como o próprio nome já indica, é a sessão para entrada do intervalo de tempo.



Estação e Satélite

 A segunda sessão é a entrada de dados necessários para selecionar os satélites que serão visíveis no gráfico a ser gerado (GPS, GLONASS, BEIDOU, etc.)


 Também apresenta os filtros de Atributo, Estação e Máscara de elevação.



Estilos de linhas

 Por fim, a última sessão é utilizada para personalização das linhas do gráfico.


 O gráfico gerado irá apresentar os dados correspondentes à pesquisa, caso não haja dados disponíveis ou alguma entrada estiver errada será mostrado um alerta.

Baixar o Aplicativo

O aplicativo está disponível para dispositivos Android versão 8.0+.
